### Getting Started

Clone the repo and navigate to the project root directory:

```
$ git clone https://github.com/jarasmar/hfh_corona_game
$ cd hfh_corona_game
```

Run the project from the terminal:
```
$ http-server
visit http://localhost:8080/
```
